# AgriMart

AgriMart is a MERN Stack agricultural marketplace platform that connects farmers, retailers, wholesalers, and customers.

## Tech Stack

### Frontend
- React.js
- Vite
- Tailwind CSS
- Axios

### Backend
- Node.js
- Express.js

### Database
- MongoDB Atlas
- Mongoose

## Features

- B2B / B2C Mode
- Product Categories
- Dynamic Product Listing
- MongoDB Integration
- Responsive Design
- REST APIs
